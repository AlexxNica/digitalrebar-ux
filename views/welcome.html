child_friendly Getting Started The simplest way to start using Digital Rebar is to use the Workloads wizards to deploy pre-built applications or operating systems; however, before you can take those actions you must either have discovered servers provisioner or created a provider using cloud credentials. Generally, it's easier to start by adding a provider. Adding a provider requires that you have an account and access key for one of the supported clouds.
